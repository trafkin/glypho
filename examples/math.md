# Testing Markdown math support

---

## Basic LaTeX Syntax in Markdown

### Equations

Inline equation using LaTeX:

```
$\( E = mc^2 \)$
```

Render as: $\( E = mc^2 \)$

---

### Greek Letters and Special Symbols

Using LaTeX:

```
$\( \alpha, \beta, \gamma \)$
$\( e, i, \pi \)$
$\( \infty, \partial, \int \)$
```

Render as:

$ \( \alpha, \beta, \gamma \)$

$ \( e, i, \pi \)$

$ \( \infty, \partial, \int \)$

---

### Matrices

Using LaTeX:


Render as:

$\begin{bmatrix}a & b\\ c & d\end{bmatrix}$ 

---

### Fractions and Roots

Using LaTeX:

```
$\( \frac{1}{2} \)$
$\( \sqrt{x^2 + y^2} \)$
```

Render as:

- $( \frac{1}{2} \)$
- $( \sqrt{x^2 + y^2} \)$
